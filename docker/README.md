## Mangosale Docker

#### Build

```js
docker build -t mangosale/ui1:v1 .
```



#### RUN

```
docker run -p8080:80 mangosale/ui1:v1
```



#### Check

```
http://localhost:8080/
```



### config

###### Dockerfile

```
FROM nginx:1.15.2-alpine
COPY ./dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
```
